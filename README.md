# 东东么么哒
## 简介
个人论坛
2017年9月开工，到2017年11月停止，2018年2月23号重新启动项目
## 技术栈
Vue2 + Koa2 + MongoDB
## 项目启动

1. 启动本地 redis
1. 启动本地 mongodb
1. 服务器启动  npm run start-server
1. vue启动  npm run dev

本次使用的 **redis** 的配置

    host为localhost
    端口号为默认的6379
    数据库为7

本次使用的 **mongodb** 的配置

    数据库 dongdongmemeda

目前所有数据表

    用户数据表  users
    博客数据表  blogs
    评论数据表  dynamics
    邀请码数据表  yaoqings
